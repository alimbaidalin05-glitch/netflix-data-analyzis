# Netflix Data Analysis

## Описание проекта

Проект посвящен анализу датасета Netflix Movies and TV Shows.

В работе выполнены:

- исследовательский анализ данных (EDA);
- очистка данных;
- визуализация данных;
- обучение модели машинного обучения;
- оценка качества модели.

## Используемые технологии

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Структура проекта

- 01_eda.ipynb — исследовательский анализ данных;
- 02_ml_analysis.ipynb — машинное обучение;
- netflix_titles.csv — исходный датасет;
- netflix_titles_clean.csv — очищенный датасет.
